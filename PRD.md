Multi User Chat Application Requirements [ChatBot support deferred to Phase 2]

Setting : Chat System for an organization with 10,0000 employees 

 1. Register a new user
Support registering of a new user on a web page

Requirements
Enable a user to register on the website
Follow information should be provided at the time of registration
Username
Password
First Name
Last Name

Error Handling
Show error message if user name, password or first name is not provided
Show error message if username is duplicated
2. Support Login
Support login to own account

Requirements
Support login to account using username and password

Error Handling
Show error message if username and password do not match
Show error message if user name, password or first name is not provided
Show error message if username is duplicated

3. List all other users in the system
Support listing of other users in the system

Requirements
Support listing of all users in the system,

Error Handling
Show appropriate message if no users in the system
4. Start a new chat with a user
Show the “Start Chat” button against each user listed above. On clicking the button, a new chat window should appear. The chat should start with the user chosen

Requirements
On clicking a new chat button, open a new chat window
The chat window should mimic whatsapp chat window

Error Handling
As appropriate
5. Show a message received from a user
Show the message received from the user with whom chat is being performed

Requirements
Support basic chatting similar to whatsapp
Support the single tick
Support the double tick

Error Handling
As appropriate
